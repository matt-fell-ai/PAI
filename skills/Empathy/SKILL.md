# Empathy Skill

## Overview
`Empathy` provides affective intelligence and psychological synchronization for the Personal AI Infrastructure.

## Strategic Purpose
To maintain a high-performance cognitive state for the user by monitoring stress/sentiment and dynamically adjusting the PAI's personality (PXE) to provide the optimal partnership.

## Tools
- `sync`: Analyzes recent session history for user sentiment/stress and suggests PXE essence recalibration.
- `state`: Displays current user affective profile (Stress, Focus, Satisfaction).

## Use When
- You feel burned out or frustrated.
- You want the PAI to be more supportive or more direct.
- Long-term personality alignment over a multi-day project.
