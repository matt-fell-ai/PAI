# Arbiter Skill - Information Integrity & Defense

| name | description |
| --- | --- |
| arbiter | Detects deepfakes, misinformation, and "slop" in ingested data. USE WHEN you are researching high-stakes topics or consuming external media. |

## The Key Insight
By 2030, the internet is 99% synthetic content. **Arbiter** is your "Truth Layer." It uses multi-model consensus and cryptographic verification to tell you what is real and what is a manipulation attempt.

## Usage

### Verify Content
```bash
pai run Arbiter verify "news_article_url"
```

### Analyze Audio
```bash
pai run Arbiter sense "voice_memo.mp3"
```

## Strategic Value
- **Deepfake Immunity**: Protects your revenue engine from fraudulent leads or fake market trends.
- **Signal-to-Slop**: Automatically filters out AI-generated filler, leaving only high-density information.
- **Trust Sovereignty**: You decide what to believe, backed by deterministic evidence.
