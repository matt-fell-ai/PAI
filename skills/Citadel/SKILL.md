# Citadel Skill - Asset & Identity Sovereignty

| name | description |
| --- | --- |
| citadel | Manages decentralized assets, cold-storage keys, and your digital "Fortress." USE WHEN you are managing high-value assets or updating security protocols. |

## The Key Insight
In 2030, your digital identity is your most targetable asset. **Citadel** is the heavy-armor for your PAI. It manages local encryption, off-chain keys, and hardware security module (HSM) integrations.

## Usage

### Audit Assets
```bash
pai run Citadel audit
```

### Lock Vault
```bash
pai run Citadel secure
```

## Strategic Value
- **Asset Sovereignty**: Manage your financial future without relying on vulnerable 3rd-party platforms.
- **Identity Defense**: Hardens your PAI against "Identity Takeover" attempts.
- **Privacy Citadel**: Ensures the "Soul" of your PAI is never leaked.
